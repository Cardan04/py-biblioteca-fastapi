# FastAPI Biblioteca 
Uma API simples para gerenciamento de livros usando **FastAPI**, **JSON local** e rotas REST básicas.  
Ideal para estudos, aprendizado ou como base para projetos maiores.

---

## Funcionalidades

✔ Listar todos os livros  
✔ Listar livro por índice  
✔ Retornar livro aleatório  
✔ Adicionar novos livros  
✔ Persistência em arquivo `books.json`  
✔ IDs únicos automáticos com `uuid4()`  

---

## Tecnologias Utilizadas

- **Python 3.10+**
- **FastAPI**
- **Uvicorn**
- **Pydantic**
- **JSON** (para persistência)
